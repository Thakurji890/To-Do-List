# 📇 Contact Dashboard

A simple and interactive **Contact Dashboard Web Application** built using **HTML, CSS, and JavaScript**. This project combines a **contact form** and a **to-do list** into a single responsive interface.

---

## 🚀 Features

### 📝 Contact Form

- Collects user data: **Name, Email, Message**
- Basic validation handling
- Displays error messages for invalid inputs

### ✅ To-Do List

- Add tasks dynamically
- Display tasks instantly
- Lightweight and fast interaction

### 🎨 UI/UX

- Clean and minimal design
- Card-based layout
- Responsive structure (works on mobile & desktop)

---

## 🛠️ Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling & Layout
- **JavaScript (Vanilla JS)** – Functionality

---

## ⚙️ How to Run

1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in your browser
